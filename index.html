Sovereign Org Profiles

An open, composable way for organizations to publish the information that defines them — once, at a URL they control — so any network or agent can discover and map it without a platform capturing the data.

See the repository, the README, and the plain-language on-ramp Get listed.

Live profiles

These are the canonical, fetchable JSON profiles served from this site. Each URL is what you register with the Murmurations Index Updater — the index stores the URL and a hash, then fetches the profile from here.
